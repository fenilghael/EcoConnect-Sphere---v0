3. Manual Testing Steps

  1. Go to http://localhost:3000
  2. Click "Login / Register" button
  3. Login with admin credentials above
  4. Look for "Welcome, System Administrator" in top-right
  5. Click the "Logout" button (should show "Logging out..." briefly)
  6. Verify you're logged out (button should change back to "Login / Register")

4. Troubleshooting

If logout doesn't work:

  • Check browser console for errors (F12)
  • Make sure backend is running on port 5000
  • Check if token is cleared from localStorage
  • Try hard refresh (Ctrl+Shift+R)
